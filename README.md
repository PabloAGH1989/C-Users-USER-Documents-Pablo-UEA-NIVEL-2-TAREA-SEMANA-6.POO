# Sistema de Gestión de Restaurante

Proyecto desarrollado en Python utilizando Programación Orientada a Objetos (POO).

El sistema permite registrar platillos y bebidas, además de mostrar el menú del restaurante mediante un menú interactivo.

## Funcionalidades

- Registrar platillos.
- Registrar bebidas.
- Mostrar menú.
- Salir del sistema.
